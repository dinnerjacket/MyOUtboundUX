# Outbound Machine

AI-powered outbound email sequence generator. Build personalized multi-step email campaigns that understand your product and craft sequences tailored to each prospect's title, industry, and company.

## Features (Phase 1)

- **Product Knowledge Base** — Upload pitch decks, case studies, and product docs. Fill in structured context (ICP, differentiators, proof points, objection handling, tone). Everything persists in localStorage.

## Coming Soon

- Prospect CSV ingestion with enrichment
- AI-powered sequence generation via Claude API
- A/B variant engine
- Instantly / Lemlist export
- Performance analytics & auto-refinement

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Push to GitHub → connect to Vercel → auto-deploys on every push.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
